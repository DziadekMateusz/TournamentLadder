# TournamentLadder
Tournament ladder simulator written in C# using Windows Forms App (.NET Framework) and hardware controller in C using Arduino IDE. Winforms app connects to the avaliable serial port (which in this project has Arduino Uno R3 plugged into it) and randomly selects 2 starting contestants (2 photos from the "Pictures" folder). User then, using 2 buttons on the breadboard which are connected to the analog pins A0 and A1 on the Arduino, may choose which one shall go into the next round and which one will be eliminated. There must be at least 2 contestants (2 photos in folder) and they will not be repeated in the following rounds. There are additional digital buttons in Winforms app to further test data exchange through serial port comunication - depending on which button is pressed, the information displayed on the LCD screen connected to the Arduino changes. Project designed and created during communication interfaces lessons at university.
